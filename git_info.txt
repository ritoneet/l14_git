1) инииализация git / редко, только когда создается проект с нуля
git init

2) просмотр статуса файлов / очень часто, всегда когда ходим увидить изменния файлов
git status

3) внести файл, в котором будет отслеживаться изменения / всегда
git add git_info.txt
git add --all

4)
git commit -m "add git_info.txt and main.py"
